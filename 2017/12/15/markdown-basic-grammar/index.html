1.标题

语法：#后跟1个空格后，再写标题，支持六级标题。
示例：

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

2.文本样式

2.1 加粗

语法：要加粗的文字左右分别用两个*号包起来。
示例：
这是加粗的文字

2.2 斜体

语法：要倾斜的文字左右分别用一个号包起来。
示例： 这是倾斜的文字*

2.3 斜体加粗

语法：要倾斜和加粗的文字左右分别用三个号包起来。
示例：
**这是斜体加粗的文字***

2.4 删除线

语法：要加删除线的文字左右分别用两个~号包起来
示例：
这是加删除线的文字

2.5 引用

语法：在引用的文字前加>即可。引用也可以嵌套，如加两个>>三个>>>n个。

示例：

这是引用的内容

这是引用的内容

这是引用的内容

2.6 分割线

语法：三个或者三个以上的 - 或者 * 都可以。

2.7 超链接

语法：[超链接名](超链接地址 "超链接title")
注：title可加可不加
示例:
百度

语法：段落之间空一行

2.9 换行符

语法：一行结束时输入两个空格

4.列表

4.1 无序列表

语法：无序列表用 - + * 任何一种都可以。
示例：

  • 列表内容
  • 列表内容
  • 列表内容

    4.2 有序列表

    语法：数字加点。
    示例：
    1.列表内容
    2.列表内容
    3.列表内容

5.表格

语法：
空行
|表头|表头|表头|
|:---|:--:|---:|
|内容|内容|内容|
|内容|内容|内容| 

注：
1）表格的语句首行上面必须为空行，不然表格不生效。
2）|、-、:之间的多余空格会被忽略，不影响布局。
3）默认标题栏居中对齐，内容居左对齐。
4）-:表示内容和标题栏居右对齐，:-表示内容和标题栏居左对齐，:-:表示内容和标题栏居中对齐。
5）内容和|之间的多余空格会被忽略，每行第一个|和最后一个|可以省略，-的数量至少有一个。

示例:

姓名 技能 排行
刘备 哭 大哥
关羽 打 二弟
张飞 骂 三弟

6.代码

语法：
单行代码：代码之间分别用一个反引号包起来
示例：
create table student
代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行
示例： 

function fun(){
    alert "弹弹弹，弹走鱼尾纹。";
}
